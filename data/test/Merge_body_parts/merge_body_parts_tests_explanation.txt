Vrste za testiranje različitih body dijelova: 

- 1. Različita ALT polja, a sve ostalo isto: 1. 
ALT polja mogu biti: A,C,G,T,N,*
+ 1. 1. jedan fajl: T, drugi fajl: A + 
+ 1. 2. jedan fajl: N, drugi fajl: A + 
1. 3. jedan fajl: *, drugi fajl: N
1. 4. jedan fajl: *, drugi fajl: A
1. 5. jedan fajl: TT, drugi fajl: A
1. 6. jedan fajl: AT, drugi fajl: T
1. 7. jedan fajl: AT, drugi fajl: A


- 2. Sve isto, a različiti ID-jevi
ID polje može biti ., ili neki string/broj
+ 2. 1. jedan fajl: ., drugi fajl: SNP1-52418888 (jedinstven za oba dokumenta)
+ 2. 2. jedan fajl: ., drugi fajl: 12 (koji se nalazi na još nekoj poziciji, na nekom hromozomu, u prvom fajlu)
2. 3. jedan fajl: 12, drugi fajl: 12
2. 4. jedan fajl: 12 na dvije pozicije u istom fajlu
2. 5. jedan fajl: rs48888888, drugi fajl: SNP1-52418888 (jedinstveni u oba fajla)
2. 6. jedan fajl: rs48888888, drugi fajl: SNP1-52418888 (u prvom ima duplikat ID, u drugom je ID jedinstven)
2. 7. jedan fajl: rs48888888, drugi fajl: SNP1-52418888 (duplikati prvi ID postoji još na najmanjem jednom mjestu u prvom fajlu, a drugi ID postoji na još najmanje jednom mjestu u drugom fajlu)
2. 8. jedan fajl: rs48888888, drugi fajl: SNP1-52418888 (prvi ID postoji u drugom fajlu na drugom mjestu, drugi ID je jedinstven za oba fajla)

- 3. Različiti REF, sve ostalo isto 
+ 3. 1. jedan fajl: C, drugi fajl: A, ALT je T
+ 3. 2. jedan fajl: C, drugi fajl: A, ALT je *

- 4. Različiti REF, različita ALT, sve ostalo isto
+ 4. 1. jedan fajl: C, *, drugi fajl: A, T 
+ 4. 2. jedan fajl: C, N, drugi fajl: A, T

- 5. Različiti FILTER
FILTER može biti: LowEVS, NOT_POLY_IN_1000G, id5, ., PASS
+ 5. 1. jedan fajl: LowEVS, drugi fajl: NOT_POLY_IN_1000G
+ 5. 2. jedan fajl: LowEVS, drugi fajl: id5
5. 3. jedan fajl: LowEVS, drugi fajl: .
5. 4. jedan fajl: LowEVS, drugi fajl: PASS
5. 5. jedan fajl: NOT_POLY_IN_1000G, drugi fajl: id5
5. 6. jedan fajl: NOT_POLY_IN_1000G, drugi fajl: .
5. 7. jedan fajl: NOT_POLY_IN_1000G, drugi fajl: PASS
5. 8. jedan fajl: id5, drugi fajl: .
5. 9. jedan fajl: id5, drugi fajl: PASS
5. 10. jedan fajl: ., drugi fajl: PASS 

- 6. Ista polja REF i ALT u jednom fajlu
+ 6. 1. jedan fajl: C, C
+ 6. 2. jedan fajl: N, N
6. 3. jedan fajl: *, *

+ 7. dva puta jedna te ista linija u jednom fajlu

+ 8. Različit INFO 
8. 1. jedan fajl: SOMATIC;QSI=41;TQSI=1;NT=het;QSI_NT=41;TQSI_NT=1;SGT=het->ref;MQ=59.85;MQ0=0;RU=T;RC=2;IC=3;IHP=11;SomaticEVS=0.00
		drugi fajl: SOMATIC;QSS=103;TQSS=1;NT=het;QSS_NT=103;TQSS_NT=1;SGT=CT->CC;DP=211;MQ=59.85;MQ0=0;ReadPosRankSum=2.22;SNVSB=0.00;SomaticEVS=0.00
